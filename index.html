Instructions:
  • Add State. Double click on the canvas to add a new state.
  • Change State Name. Click to select a state, and then type to change its name.
  • Start, End State. Double click on an existing state to toggle between Start, Final and Regular state types.
  • Add Transition. Shift-drag between two states to add a transition.
  • Add Transition Symbol. Click to select a transition and type to set its transition symbol (separate by comma for more than one symbol).
  • Delete something. Click to select an item an press the Delete key or Shift-Backspace to delete it.
  • Epsilon. Type \e to write epsilon. All numbers are drawn as subscripts.
  • Simulate. Type a word and click simulate to run a word evaluation. (Simulation only works for DFAs, no ε-transitions.)
